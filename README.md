# ezee
anthony,jacob,evan

##  Description
it uses random to pick random number 1-6 on a die and then you reroll

###  Flowchart
```mermaid
graph TD;
  main-->first_roll;
  main-->reroll_many;
  main-->output_dice;
  first_roll--roll;
  find_mode-->count_frequency
  main-->list_unmatched_dice
  reroll_many-->reroll_1
  reroll_1--> roll
  reroll_many-->find_mode
```
